# How to use
1.pythonをダウンロードします。3.10らへんがおすすめ。

2.ライブラリは探してインストールして()

3.python preview.pyで実行。

# どうやって使うん？
1.LOGO.imgやbinをLOGO BUILDER

https://xdaforums.com/t/changing-boot-logo-logo-bin-program.1953726/

を使ってプロジェクトとして作成します。

2.それでできたプロジェクトフォルダーを本ソフトウェアで開きます。

3.あとは座標等を調整して完了です。

4.logoを改変する際の位置調整などにご活用ください
