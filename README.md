# LOGO.IMG-PREVIEWER
logo.imgをプレビュー再生するpythonスクリプト

# この作品は個人利用でのりようにとどめてください！
## 商用利用、二次配布や自作発言、改変しての配布や利用は禁止です！

この作品の初期状態はtaba04br3のlogo.imgに最適化されています。

使用する際は自分で下のスライダーを使い座標を調整してください。

taba04br3のlogo.imgは本プロジェクトのoriginalフォルダーにてダウンロード可能です

# How to use
1.pythonをダウンロードします。3.10らへんがおすすめ。

(事前にこのgithubからpreview.pyをダウンロードしといてね。releaseには置いてないよ)

2.ライブラリは探してインストールして()

3.python preview.pyで実行。

# どうやって使うん？
1.LOGO.imgやbinをLOGO BUILDER

https://xdaforums.com/t/changing-boot-logo-logo-bin-program.1953726/

を使ってプロジェクトとして作成します。

2.それでできたプロジェクトフォルダーを本ソフトウェアで開きます。

3.あとは座標等を調整して完了です。

4.logoを改変する際の位置調整などにご活用ください
